# 京津冀 PM2.5 预测与气象贡献度展示

这是一个面向公网部署的 Streamlit 交互网页，用于展示“基于机器学习的京津冀 PM2.5 浓度预测与气象贡献度分析”课题成果。网页内嵌训练后的 LightGBM 模型，可选择城市、日期、小时和模型口径，输入气象条件后预测 PM2.5 浓度，并展示模型指标、SHAP 贡献、疫情分时期对比、v2 气象贡献模型和预测台内的气象背景图。

## 公网访问

在线网页地址：

https://bth-pm25-ai.streamlit.app/

## 项目概况

- 研究区域：京津冀 13 个城市。
- 数据时间：2018-01-01 00:00 至 2026-05-31 18:00。
- 主训练表：2018 年以来 PM2.5、气象因子、ERA5 边界层高度 PBLH 与特征工程结果。
- 样本规模：958,711 条 city-hour 记录。
- PM2.5 缺失率：0%。
- ERA5 PBLH 缺失率：0%。
- 疫情分期：疫情前 2018-2019、疫情期 2020-2022、疫情后 2023+。

说明：2018-2022 PM2.5 使用 CNEMC/quotsoft 城市小时空气质量数据；2023+ 使用此前整合的数据源。跨时期解释时应注明数据源存在差异。

## 网页功能

- 预测台：选择城市、日期、小时和模型，输入温度、湿度、风速、风向、气压、PBLH、逆温、PM2.5 时滞等变量，确认后输出当前小时 PM2.5 预测结果，并在同页展示城市历史 PM2.5/PBLH、季节参考、风场和当天气象变量剖面。
- 模型介绍：展示全时期高精度模型、分时期高精度模型、旧版气象归因模型和 v2 气象贡献模型的指标、用途、特征口径、SHAP 贡献和残差结果。
- 训练策略：按“高精度预测模型”和“气象归因模型”分别说明变量准入、时间切分、特征生成、LightGBM/Optuna 调参、目标变量选择和评估口径。
- 空气质量配色：优/良好为绿色，轻度污染为黄色，中度污染及更差为红色至深红色。

## 模型结果

| 模型 | 训练口径 | 测试 MAE | 测试 RMSE | 测试 R2 |
|---|---|---:|---:|---:|
| 全时期高精度预测模型 | 2018+ extended | 7.72 | 12.06 | 0.917 |
| 疫情前高精度预测模型 | 2018-2019 extended | 2.27 | 3.60 | 0.993 |
| 疫情前气象贡献模型 | 2018-2019 meteorology | 25.06 | 34.43 | 0.322 |
| 疫情期高精度预测模型 | 2020-2022 extended | 1.77 | 3.09 | 0.993 |
| 疫情期气象贡献模型 | 2020-2022 meteorology | 15.88 | 26.48 | 0.501 |
| 疫情后高精度预测模型 | 2023+ extended | 1.33 | 2.59 | 0.997 |
| 疫情后气象贡献模型 | 2023+ meteorology | 18.79 | 26.84 | 0.691 |
| 疫情前气象贡献 v2 模型 | 2018-2019 meteorology-only, log1p(PM2.5) | 19.47 | 27.98 | 0.552 |
| 疫情期气象贡献 v2 模型 | 2020-2022 meteorology-only, PM2.5 原值 | 13.31 | 21.40 | 0.674 |
| 疫情后气象贡献 v2 模型 | 2023+ meteorology-only, 气候态异常 | 17.61 | 24.92 | 0.734 |

高精度模型主要用于预测展示，允许使用 PM2.5 时滞、滚动均值和共污染物变量；气象贡献模型用于学术归因，控制污染持续性和共污染物后，重点考察温湿度、气压、风输送、逆温、PBLH 和复合扩散条件对 PM2.5 的独立解释力。

v2 气象贡献模型在旧版气象归因模型基础上扩展为 248 个气象特征，加入 PBLH、风速、U/V 风分量、相对湿度、气压、逆温指数的多时滞与滚动统计，并构造通风系数、连续低 PBLH、连续弱风、连续高湿、南风输送、北风清洁输送等气象过程指标。该版本仍保持 meteorology-only 变量准入原则，不使用 PM2.5 时滞、滚动均值和共污染物，因此更适合支撑“大气扩散条件与污染浓度变化关系”的学术讨论。

## 部署

公网地址：

```text
https://bth-pm25-ai.streamlit.app/
```

Streamlit Cloud 部署入口：

```text
streamlit_app.py
```

本地运行：

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

部署包包含：

```text
app/                 Streamlit 应用源码
app_assets/          网页展示用指标、SHAP、城市画像和历史数据
models/              已训练 LightGBM 模型
streamlit_app.py     云平台入口
requirements.txt     运行依赖
```

完整原始数据和下载脚本不放入本部署仓库；网页运行只依赖模型文件和轻量展示资产。
