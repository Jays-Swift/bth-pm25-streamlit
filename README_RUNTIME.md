# 运行目录说明

该目录是交互式研究界面的轻量运行目录，包含研究界面入口、模型对象、城市画像、模型指标和 SHAP 解释结果。完整原始数据和下载脚本保留在项目根目录中，不放入该轻量运行目录。

## 内容

- `streamlit_app.py`：运行入口。
- `app/`：研究界面源码。
- `models/`：已训练模型对象。
- `app_assets/`：城市画像、指标和 SHAP 表。

## 研究口径

当前运行目录对应 2018-2026 京津冀 PM2.5 与气象特征数据，包含全时期高精度预测模型、疫情前/疫情期/疫情后三个阶段的分时期高精度模型，以及分时期气象贡献模型。界面中空气质量等级配色为：优/良为绿色，轻度污染为黄色，中度污染及更高等级为红色至深红色。

## 本地运行

```bash
streamlit run streamlit_app.py
```
